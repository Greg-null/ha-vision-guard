# Home Assistant Add-on: Vision Guard

Lokale Objekterkennung (YOLOv8) mit Bounding Boxes und GIF-Erstellung aus Kamera-Aufnahmen.

## Installation

1. In Home Assistant: **Settings → Add-ons → Add-on Store → Repositories**
2. Repository-URL hinzufügen: `https://github.com/Greg-null/ha-vision-guard`
3. **Vision Guard** installieren
4. Addon starten

## Nutzung

Das Addon stellt eine REST API auf Port 5000 bereit. Siehe [Dokumentation](vision-guard/DOCS.md).
